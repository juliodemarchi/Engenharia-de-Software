# 🎓 Engenharia de Software - Portfólio de Estudos (Estácio)

Este repositório centraliza todos os meus projetos, exercícios e anotações teóricas realizados durante a licenciatura em Engenharia de Software. O objetivo é documentar a minha evolução técnica e organizar o conhecimento por tecnologias e paradigmas.

---

## 📂 Organização do Repositório

Cada pasta abaixo representa uma trilha de aprendizagem específica dentro da grade curricular:

### 🦀 [C-Language](./C-Language/)
* **Descrição:** Foco em lógica de programação estruturada e fundamentos de computação.
* **Conteúdo:** Manipulação de variáveis, estruturas de repetição, ponteiros e gestão de memória.
* **Contexto:** Disciplinas iniciais e base do CS50.

### 🏛️ [CPP-DataStructures](./CPP-DataStructures/)
* **Descrição:** Implementação de estruturas de dados clássicas utilizando C++.
* **Conteúdo:** Listas ligadas, pilhas, filas, árvores binárias e algoritmos de ordenação.
* **Contexto:** Disciplinas de Estrutura de Dados.

### 🐍 [Python](./Python/)
* **Descrição:** Desenvolvimento com foco em produtividade, POO e scripts de automação.
* **Conteúdo:** Sintaxe moderna, Programação Orientada a Objetos (POO) e integração com bibliotecas de análise de dados.
* **Contexto:** Desenvolvimento de software e algoritmos avançados.

### ☕ [Java-Development](./Java-Development/)
* **Descrição:** Desenvolvimento de aplicações robustas e escaláveis.
* **Conteúdo:** Padronização de projetos (Design Patterns), interfaces, herança e desenvolvimento Web com frameworks.
* **Contexto:** Programação Orientada a Objetos e Sistemas Distribuídos.

### 🌐 [JavaScript](./JavaScript/)
* **Descrição:** Tecnologias para o ecossistema Web (Frontend e Backend).
* **Conteúdo:** Manipulação do DOM, lógica assíncrona, ES6+ e introdução a Node.js.
* **Contexto:** Desenvolvimento Web em camadas.

### 🗄️ [SQL-Databases](./SQL-Databases/)
* **Descrição:** Modelagem e manipulação de dados em bases de dados relacionais.
* **Conteúdo:** DDL (Data Definition Language), DML (Data Manipulation Language), Joins e Normalização.
* **Contexto:** Disciplinas de Modelação de Dados e SQL.

### ☁️ [Cloud-Computing](./Cloud-Computing/)
* **Descrição:** Estudos sobre infraestrutura, virtualização e serviços em nuvem.
* **Conteúdo:** Conceitos de SaaS, PaaS, IaaS e deploy de aplicações.
* **Contexto:** Engenharia de Requisitos e Sistemas Distribuídos.

### 🚀 [Projetos-Estacio](./Projetos-Estacio/)
* **Descrição:** Projetos transversais e atividades de extensão obrigatórias.
* **Conteúdo:** Projetos práticos que unem diversas disciplinas e entregas de portfólio.
